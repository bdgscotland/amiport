# test-fsemu-visual-cases.txt -- Visual verification tests for mg 3.7
# Runs in a SEPARATE FS-UAE pass from functional tests (ADR-024).
# Uses SCRAPE + EXPECT_AT to verify screen content after keyboard input.
# Requires forked FS-UAE with per-unit ANSI capture.
#
# NOTE: CMD_WRITE captures static content display (file load, help text)
# but NOT interactive echo (typed characters, cursor movement). Interactive
# rendering verification needs the ADR-025 custom trap approach.

# ============================================================
# Visual: file content correctly displayed on open
# ============================================================

ITEST: Visual: file content displayed correctly on open
LAUNCH: WORK:mg -n WORK:test-mg-basic.txt
KEYS: WAIT3000
SCRAPE
EXPECT_AT 1,1,Hello, Amiga world!
EXPECT_AT 2,1,This is a basic test file for mg.
EXPECT_AT 3,1,Line three has some text.
EXPECT_AT 4,1,Line four: the quick brown fox jumps over the lazy dog.
EXPECT_AT 5,1,Line five: testing cursor movement and search.

# ============================================================
# Visual: scroll file displays from correct starting line
# ============================================================

ITEST: Visual: scroll file displays all lines correctly
LAUNCH: WORK:mg -n WORK:test-mg-scroll.txt
KEYS: WAIT3000
SCRAPE
EXPECT_AT 1,1,Line 001: The Amiga 1200 was released in 1992.
EXPECT_AT 3,1,Line 003: The AGA chipset brought 256 colours on screen.
EXPECT_AT 5,1,Line 005: AmigaOS uses a microkernel called Exec.
