# Conceptual Idea Records (CIDR)

This directory contains Conceptual Idea Records for the amiport project.

CIDRs capture early-stage ideas, explorations, and hypotheses that haven't yet solidified into product decisions (PDRs) or architecture decisions (ADRs). They're a holding area for thinking — some will graduate into formal decisions, others will be shelved or rejected.

CIDRs are intentionally lightweight. They encourage writing things down early rather than losing ideas.
