# ILBM.CLUT.doc / Introduction

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This memo describes the IFF supplement for the new chunk "CLUT".

Description:

A CLUT (Color Look Up Table) is a special purpose data module containing
table with 256 8-bit entries.  Entries in this table can be used directly
as a translation for one 8-bit value to another.
