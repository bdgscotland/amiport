# 2 Audio Device / Allocation and Arbitration Commands

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These commands allow the audio channels to be shared among different tasks
and programs. None of these commands can be called from interrupt code.
