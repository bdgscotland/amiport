# 7 / Amiga Reset Handling / Removing A Reset Handler (KBD_REMRESETHANDLER)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This command is used to remove a keyboard reset handler from the system.
You need to supply the same [Interrupt](autodocs-2.0/includes-exec-interrupts-h.md) structure to this command that you
used with the [KBD_ADDRESETHANDLER](autodocs-2.0/keyboard-device-kbd-addresethandler.md) command.
