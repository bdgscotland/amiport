# Standard Data and Property Chunks / End-Mark SEvent

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The [SEvent](devices/standard-data-and-property-chunks-data-chunk-trak.md) type SID_Mark is reserved for an end marker in working memory.
This event is never stored in a file.  It may be useful if you decide to
use the filed [TRAK](devices/standard-data-and-property-chunks-data-chunk-trak.md) format intact in working memory.
