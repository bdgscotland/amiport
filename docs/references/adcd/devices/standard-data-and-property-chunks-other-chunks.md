# Standard Data and Property Chunks / Other Chunks

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Issue: In the future, we may define an optional chunk containing Fourier
series coefficients for a repeating waveform.  An editor for this kind of
synthesized voice could modify the coefficients and regenerate the
waveform.

See the IFF Registry and the Third-Party Specification section for details
on additional 8SVX Chunks such as CHAN, PAN, SEQN and FADE.
