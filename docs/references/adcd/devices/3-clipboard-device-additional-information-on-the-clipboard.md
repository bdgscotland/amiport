# 3 Clipboard Device / Additional Information on the Clipboard Device

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Additional programming information on the clipboard device can be found in
the include files for the clipboard device, iffparse library and utility
library, and the Autodocs for all three. They are contained in the
Amiga ROM Kernel Reference Manual: Includes and Autodocs.
