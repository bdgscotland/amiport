# HEAD.doc / FORM

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

FORM ID:  HEAD

FORM Description:

FORM HEAD is the file storage format of the Flow idea processor by New
Horizons Software, Inc.  Currently only the TEXT and NEST chunks are used.
There are plans to incorporate FSCC and some additional chunks for headers
and footers.
