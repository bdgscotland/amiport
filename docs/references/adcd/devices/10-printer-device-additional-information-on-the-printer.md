# 10 Printer Device / Additional Information on the Printer Device

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Additional programming information on the printer device can be found in
the include files and the Autodocs for the printer device.   Both are
contained in the Amiga ROM Kernel Reference Manual: Includes and Autodocs.



```c
                       Printer Device Information
                    ---------------------------------
                    INCLUDES        devices/[printer.h](autodocs-2.0/includes-devices-printer-h.md)
                                    devices/[printer.i](autodocs-2.0/includes-devices-printer-i.md)
                                    devices/[prtbase.h](autodocs-2.0/includes-devices-prtbase-h.md)
                                    devices/[prtbase.i](autodocs-2.0/includes-devices-prtbase-i.md)
                                    devices/[prtgfx.h](autodocs-2.0/includes-devices-prtgfx-h.md)
                                    devices/[prtgfx.i](autodocs-2.0/includes-devices-prtgfx-i.md)

                    AUTODOCS        [printer.doc](autodocs-2.0/printer-doc.md)
```
Additional printer drivers can be found on Fred Fish Disk #344 under
RKMCompanion.
