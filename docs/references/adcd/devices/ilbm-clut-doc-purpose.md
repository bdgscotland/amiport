# ILBM.CLUT.doc / Purpose

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

To store 8-bit data look up tables in a simple format for later retrieval.
These tables are used to translate or bias 8-bit intensity, contrast,
saturation, hue, color registers, or other similar data in a reproducable
manner.
