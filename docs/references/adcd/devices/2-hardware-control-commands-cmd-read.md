# 2 / Hardware Control Commands / CMD_READ

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

[CMD_READ](autodocs-2.0/audio-device-cmd-read.md) is a single-channel command. Its only function is to return a
pointer to the current [CMD_WRITE](devices/2-hardware-control-commands-cmd-write.md) command. It enables you to determine
which request is being processed.
