# Standard Data and Property Chunks / Instant Music Private SEvents

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Sixteen [SEvent](devices/standard-data-and-property-chunks-data-chunk-trak.md)s are used for private data for the Instant Music program.
SID values 144 through 159 are reserved for this purpose.  Other programs
should skip over these SEvents.
