# Standard Data and Property Chunks / Set MIDI Preset SEvent

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The [SEvent](devices/standard-data-and-property-chunks-data-chunk-trak.md) SID_MIDI_Preset is for recorder programs to record the
set-MIDI-preset low level event.  The data byte contains a MIDI preset
number. Other programs should use [instrument registers](devices/background-instrument-registers.md) instead.
