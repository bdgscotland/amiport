# Standard Data and Property Chunks / Future Properties

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

New optional property chunks may be defined in the future to store
additional formatting information.  They will be used to represent
formatting not encoded in standard ISO/ANSI control sequences and for
"inherited" formatting in structured documents.  Text orientation might be
one example.
