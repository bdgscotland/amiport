# DGetChar

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

NAME

    DGetChar - get a character from the parallel port
SYNOPSIS

```c
    char = DGetChar()
    D0
```
FUNCTION

    get the next character from the parallel port.
