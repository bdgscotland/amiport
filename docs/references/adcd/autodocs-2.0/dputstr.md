# DPutStr

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

NAME

    DPutStr - put a string to the parallel port
SYNOPSIS

```c
    DPutStr(string)
           A0
```
FUNCTION

    put a null terminated string to the parallel port.
