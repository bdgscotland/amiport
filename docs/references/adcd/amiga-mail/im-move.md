# IM_MOVE

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The IM_MOVE method erases and redraws an image.  It is intended for use in
some subclass of imageclass that performs animation or smooth dragging.
Currently, no public boopsi classes implement this method.
