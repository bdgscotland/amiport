# Starting the ShowNote Server

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The ShowNote server should run as a backround CLI process.  To start
it, type the following line at a CLI prompt:


    run >nil: shownote
You can start up the server when you boot your Amiga by adding that
line to s:user-startup.
