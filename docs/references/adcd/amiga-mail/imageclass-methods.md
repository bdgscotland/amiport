# Imageclass Methods

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Imageclass defines several methods of its own which subclasses of
imageclass either have to implement or pass on to their superclass. The
method IDs for imageclass are defined in <intuition/imageclass.h>.  Each
method requires some parameters.  The MethodID is the only parameter
common to each method.
