# The Global Drawing Attribute Chunks

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The following chunks describe global attributes of a DR2D document.

 [DRHD (0x44524844)       /* Drawing header */](amiga-mail/drhd-0x44524844-drawing-header.md) 
 [PPRF (0x50505249)       /* Page preferences */](amiga-mail/pprf-0x50505249-page-preferences.md) 
 [CMAP (0x434D4150)       /* Color map (Same as ILBM CMAP) */](amiga-mail/cmap-0x434d4150-color-map-same-as-ilbm-cmap.md) 
 [FONS (0x464F4E53)       /* Font chunk (Same as FTXT FONS chunk) */](amiga-mail/fons-0x464f4e53-font-chunk-same-as-ftxt-fons-chunk.md) 
 [DASH (0x44415348)       /* Line dash pattern for edges */](amiga-mail/dash-0x44415348-line-dash-pattern-for-edges.md) 
 [AROW (0x41524F57)       /* An arrow-head pattern */](amiga-mail/arow-0x41524f57-an-arrow-head-pattern.md) 
 [FILL (0x46494C4C)       /* Object-oriented fill pattern */](amiga-mail/fill-0x46494c4c-object-oriented-fill-pattern.md) 
 [LAYR (0x4C415952)       /* Define a layer */](amiga-mail/layr-0x4c415952-define-a-layer.md)
