The new intuition.library provides the programmer with flexible yet
simple methods of opening Intuition screens and windows.  This article
is an introduction to some of the Intuition and relevant
graphics.library functions that make this possible. For more in depth
information see the 1990 DevCon notes as well as the Autodoc and
include files for release 2.0.
