# 10 Intuition Mouse and Keyboard / Function Reference

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The following are brief descriptions of the Intuition functions that
relate to the use of the mouse and keyboard under Intuition. See the Amiga
ROM Kernel Reference Manual: Includes and Autodocs for details on each
function call.

         Table 10-4: Functions for Intuition Mouse and Keyboard
  ____________________________________________________________________
 |                                                                    |
 |      Function                  Description                         |
 |====================================================================|
 |    [DoubleClick()](autodocs-2.0/intuition-library-doubleclick.md)  Test two time values for double click status.    |
 |--------------------------------------------------------------------|
 |     [SetPointer()](autodocs-2.0/intuition-library-setpointer.md)  Change the Intuition pointer imagery for an open |
 |                   window.                                          |
 |   [ClearPointer()](autodocs-2.0/intuition-library-clearpointer.md)  Restore the default Intuition pointer imagery.   |
 |--------------------------------------------------------------------|
 |  [SetMouseQueue()](autodocs-2.0/intuition-library-setmousequeue.md)  Change the mouse queue for an open window.       |
 |    [ReportMouse()](autodocs-2.0/intuition-library-reportmouse.md)  A function C programmers should not use.         |
 |____________________________________________________________________|
