# 5 / Proportional Gadget Type / Logical Types of Proportional Gadgets

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

There are two usual ways in which [proportional](libraries/5-intuition-gadgets-proportional-gadget-type.md) gadgets are used
(corresponding to the [scroller](libraries/15-the-kinds-of-gadtools-gadgets-scroller-gadgets.md) and [slider](libraries/15-the-kinds-of-gadtools-gadgets-slider-gadgets.md) gadgets of the GadTools
library).  The only difference between sliders and scrollers is the way
they are managed internally by the application.  The GadTools library
provides a high level interface to proportional gadgets, simplifying the
management task for these types of objects.
