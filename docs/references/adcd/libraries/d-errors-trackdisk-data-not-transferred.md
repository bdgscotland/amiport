# D / Errors / Trackdisk Data not Transferred

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Make sure your trackdisk buffers are in Chip RAM under 1.3 and lower
versions of the operating system.
