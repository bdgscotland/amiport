# B / gadgetclass / Changed Methods: OM_NEW

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This method allocates space for an embedded struct [Gadget](libraries/5-intuition-gadgets-gadget-structure.md) (defined in
<intuition/[intuition.h](autodocs-2.0/includes-intuition-intuition-h.md)>) and initializes some of the attributes defined by
[gadgetclass](libraries/b-boopsi-class-reference-gadgetclass.md).
