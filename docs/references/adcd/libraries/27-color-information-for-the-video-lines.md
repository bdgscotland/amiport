# 27 / / / Color Information for the Video Lines

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The hardware reads the system display memory to obtain the color
information for each line.  As the video display beam sweeps across the
screen producing the display line, it changes color, producing the images
you have defined.  On the current generation of Amiga hardware, there are
4,096 possible colors.
