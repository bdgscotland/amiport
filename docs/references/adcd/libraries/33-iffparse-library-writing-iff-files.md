# 33 IFFParse Library / Writing IFF Files

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

IFFParse provides facilities for writing [IFF](libraries/33-iffparse-library-the-structure-of-iff-files.md) files.  Again, IFFParse makes
no assumptions about the data you're writing, and concerns itself with
verifying the syntax of your output.
