# B / frbuttonclass / Attributes: GA_Width, GA_Height (S)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These attribute correspond to the gadget's Width and Height fields.  If
the framing image supports [IM_FRAMEBOX](libraries/b-imageclass-new-methods-im-framebox.md), changing these resizes the framing
image.  The framing image re-centers itself around the display element as
best it can, and the [frbuttonclass](libraries/b-boopsi-class-reference-frbuttonclass.md) gadget re-renders the whole itself.
