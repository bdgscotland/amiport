# B / frameiclass / Attributes: IA_EdgesOnly (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

If this attribute is TRUE, the frame does not fill itself, it just draws
its edges.
