# B / gadgetclass / Attributes: GA_Highlight (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute corresponds to the [GFLG_GADGHIGHBITS](autodocs-2.0/includes-intuition-intuition-h.md) portion of the
gadget's [Gadget.Flags](libraries/5-intuition-gadgets-gadget-structure.md) field.  This attribute can be one of four values
(from <intuition/[intuition.h](autodocs-2.0/includes-intuition-intuition-h.md)>):


```c
    [GFLG_GADGHCOMP](libraries/5-gadget-structure-gadget-flags.md), [GFLG_GADGHBOX](libraries/5-gadget-structure-gadget-flags.md), [GFLG_GADGHIMAGE](libraries/5-gadget-structure-gadget-flags.md), [GFLG_GADGHNONE](libraries/5-gadget-structure-gadget-flags.md)
```
See the "[Intuition Gadgets](libraries/5-gadget-structure-gadget-flags.md)" chapter of the Amiga ROM Kernel Reference
Manual: Libraries for more information.
