# D / Errors / Windows--Visual Problems

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Many visual problems in windows can be caused by improper font
specification or improper setting of gadget flags.  See the Appendix E on
"[Release 2 Compatibility](libraries/e-release-2-changes-that-can-affect-compatibility-intuition.md)" for detailed information on common problems.
