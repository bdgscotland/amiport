# B / sysiclass / Attributes: SYSIA_DrawInfo (I)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute contains a pointer to a [DrawInfo](libraries/3-screen-data-structures-other-screen-data-structures.md) structure (defined in
<intuition/[screens.h](autodocs-2.0/includes-intuition-screens-h.md)>) describing the target screen.  The class requires
this attribute in order to generate the image into a bitmap cache.
