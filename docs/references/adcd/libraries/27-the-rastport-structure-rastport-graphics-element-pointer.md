# 27 / / The RastPort Structure / RastPort Graphics Element Pointer

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The graphics element pointer in the [RastPort](libraries/27-drawing-routines-the-rastport-structure.md) structure is called [GelsInfo](libraries/28-about-the-gels-system-the-gels-system.md).
If you are doing graphics animation using the GELS system, this pointer
must refer to a properly initialized GelsInfo structure.  See the chapter
on "[Graphics Sprites, Bobs and Animation](libraries/28-about-the-gels-system-the-gels-system.md)" for more information.
