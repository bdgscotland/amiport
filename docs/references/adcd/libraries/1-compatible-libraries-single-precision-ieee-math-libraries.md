# 1 / / Compatible Libraries / Single Precision IEEE Math Libraries

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The [Release 2 single precision IEEE](libraries/35-math-libraries-ieee-single-precision-basic-math-library.md) math libraries are also compatible
with 1.3.  These libraries provide single-precision math functions that
will use a math coprocessor if available.
