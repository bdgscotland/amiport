# B / gadgetclass / Attributes:GA_Left, GA_Top, GA_Width, GA_Height (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These attributes correspond to the [Gadget](libraries/5-intuition-gadgets-gadget-structure.md) structure's [LeftEdge](libraries/5-intuition-gadgets-gadget-structure.md), [TopEdge](libraries/5-intuition-gadgets-gadget-structure.md),
[Width](libraries/5-intuition-gadgets-gadget-structure.md), and [Height](libraries/5-intuition-gadgets-gadget-structure.md) fields.  Setting these clears the "[gadget relative](libraries/b-ga-relright-ga-relbottom-ga-relwidth-ga-relheight-is.md)"
flags (below).
