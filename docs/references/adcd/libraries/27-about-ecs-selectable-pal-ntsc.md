# 27 / / About ECS / Selectable PAL/NTSC

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The Enhanced Chip Set can be set to NTSC or PAL modes under software
control.  Its initial default behavior is determined by a jumper or trace
on the system motherboard.  This has no bearing on Productivity mode and
other programmable scan operations, but the new system software can
support displays in either mode.
