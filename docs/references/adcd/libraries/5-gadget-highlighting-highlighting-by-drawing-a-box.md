# 5 / Gadget Highlighting / Highlighting by Drawing a Box

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

To highlight by drawing a simple border around the gadget's select box,
set the [GFLG_GADGHBOX](libraries/5-gadget-structure-gadget-flags.md) bit in the [Flags](libraries/5-intuition-gadgets-gadget-structure.md) field.
