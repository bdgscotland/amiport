# 8 Intuition Images, Line Drawing and Text / Function Reference

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The following are brief descriptions of the Intuition functions that
relate to the use of graphics under Intuition. See the Amiga ROM Kernel
Reference Manual: Includes and Autodocs for details on each function call.


         Table 8-1: Functions for Intuition Drawing Capabilities
  _______________________________________________________________________
 |                                                                       |
 |           Function                Description                         |
 |=======================================================================|
 |          [DrawBorder()](autodocs-2.0/intuition-library-drawborder.md)  Draw a border into a rast port.                |
 |           [DrawImage()](autodocs-2.0/intuition-library-drawimage.md)  Draw a image into a rast port.                 |
 |          [PrintIText()](autodocs-2.0/intuition-library-printitext.md)  Draw Intuition text into a rast port.          |
 |     [IntuiTextLength()](autodocs-2.0/intuition-library-intuitextlength.md)  Find the length of an IntuiText string.        |
 |-----------------------------------------------------------------------|
 |        [BeginRefresh()](autodocs-2.0/intuition-library-beginrefresh.md)  Begin optimized rendering after a refresh      |
 |                        event.                                         |
 |          [EndRefresh()](autodocs-2.0/intuition-library-endrefresh.md)  End optimized rendering after a refresh event. |
 |-----------------------------------------------------------------------|
 |   [GetScreenDrawInfo()](autodocs-2.0/intuition-library-getscreendrawinfo.md)   Get screen drawing information (V36).         |
 |  [FreeScreenDrawInfo()](autodocs-2.0/intuition-library-freescreendrawinfo.md)  Free screen drawing information (V36).         |
 |_______________________________________________________________________|
