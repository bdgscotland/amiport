# C Sample Library Source Code

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This appendix contains the source code to an Exec library called
sample.library.  It also includes some example code that uses the library.
The files include:

 [makefile](libraries/lib-examples-makefile.md)            A make file to build the library and examples.
 [asmsupp.i](libraries/lib-examples-asmsupp-i.md)           Assorted low level assembly support routines used by
                     the example library.
 [sample_rev.i](libraries/lib-examples-sample-rev-i.md)        The version file generated by bumprev.
 [sample_protos.h](libraries/lib-examples-sample-protos-h.md)     Prototypes for sample.library functions.
 [sample_pragmas.h](libraries/lib-examples-sample-pragmas-h.md)    Pragmas for sample.library functions.
 [samplebase.i](libraries/lib-examples-samplebase-i.md)        Definition of sample.library base.
 [samplebase.h](libraries/lib-examples-samplebase-h.md)        include file defining sample.library base.
 [sample.library.asm](libraries/lib-examples-sample-library-asm.md)  Example run-time library source code.
 [sample_lvos.asm](libraries/lib-examples-sample-lvos-asm.md)     The .fd file for the sample library.
 [sample_stubs.asm](libraries/lib-examples-sample-stubs-asm.md)    Assembler stubs for C interface to library functions.
 [clibtest.c](libraries/lib-examples-clibtest-c.md)          C example that calls the sample.library functions.
 [alibtest.asm](libraries/lib-examples-alibtest-asm.md)        Assembly code example that calls the sample.library

                     functions.
