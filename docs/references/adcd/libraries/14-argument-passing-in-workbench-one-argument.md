# 14 / / Argument Passing In Workbench / One Argument

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

A program started by activating its tool icon gets one argument in the
[WBStartup](libraries/14-workbench-environment-wbstartup-message.md) message: the name of the tool itself.
