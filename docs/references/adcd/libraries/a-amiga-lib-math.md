[fpa()](autodocs-2.0/amiga-lib-fpa.md)

```c
    Accepts an FFP number and the address of the ASCII string where its
    converted output is to be stored.  The number is converted to a NULL
    terminated ASCII string and stored at the address provided.
    Additionally, the base ten (10) exponent in binary form is returned.
```
