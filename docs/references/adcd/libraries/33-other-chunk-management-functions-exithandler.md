# 33 / / Other Chunk Management Functions / ExitHandler()

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This is used to install your own custom chunk exit handler.  See the
"[Custom Chunk Handlers](libraries/33-advanced-topics-custom-chunk-handlers.md)" section below for more information.  [StopOnExit()](libraries/33-other-chunk-management-functions-stoponexit.md)
is internally built on top of this.
