# B / fillrectclass / Changed Methods: IM_DRAW

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This method asks a [fillrectclass](libraries/b-boopsi-class-reference-fillrectclass.md) object to render itself relative to the
position ([LeftEdge and TopEdge](libraries/8-creating-images-image-structure.md)) and dimensions ([Width and Height](libraries/8-creating-images-image-structure.md)) in its
embedded [Image](libraries/8-creating-images-image-structure.md) structure.  See the [imageclass](libraries/b-boopsi-class-reference-imageclass.md) description of [IM_DRAW](libraries/b-imageclass-new-methods-im-draw.md) for
more details.
