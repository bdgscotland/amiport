# B / gadgetclass / Attributes: GA_Image (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attibute is a pointer to either a Boopsi image or a Release
1.3-compatible Intuition image.  This attribute corresponds to the
[Gadget](libraries/5-intuition-gadgets-gadget-structure.md)'s [GadgetRender](libraries/5-intuition-gadgets-gadget-structure.md) field.  The [gadgetclass](libraries/b-boopsi-class-reference-gadgetclass.md) dispatcher will not dispose
of this image when it disposes of the gadget object.
