# B / groupgclass / Attributes: GA_Left, GA_Top (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These attributes correspond to the embedded [Gadget.LeftEdge](libraries/5-intuition-gadgets-gadget-structure.md) and
[Gadget.TopEdge](libraries/5-intuition-gadgets-gadget-structure.md) fields.  Setting these attributes in [groupgclass](libraries/b-boopsi-class-reference-groupgclass.md) object
causes it to change its position as well as the position of each of the
gadgets that have been added to the group gadget.
