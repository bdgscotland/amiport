# 5 / / Gadget Structure For String Gadgets / Imagery and Highlighting

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Any type of image may be supplied for the rendering of a [string](libraries/5-intuition-gadgets-string-gadget-type.md)
gadget--image, border, or no image at all.  The highlighting for a string
gadget must be the complementing type ([GFLG_GADGHCOMP](libraries/5-gadget-structure-gadget-flags.md)).  Alternate imagery
may not be used for highlighting.
