# B / propgclass / Attributes: GA_Border (I)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

[Propgclass](libraries/b-boopsi-class-reference-propgclass.md) intercepts this [gadgetclass](libraries/b-boopsi-class-reference-gadgetclass.md) attribute to prevent gadgetclass
from setting up a border.  If an application tries to set this attribute
for a propgclass gadget, the prop gadget turns itself into an AUTOKNOB
gadget.
