# 6 Intuition Menus / Function Reference

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The following are brief descriptions of the Intuition functions that
relate to the use of Intuition menus.  See the Amiga ROM Kernel Reference
Manual: Includes and Autodocs for details on each function call.


                 Table 6-2: Functions for Intuition Menus
  _______________________________________________________________________
 |                                                                       |
 |       Function                Description                             |
 |=======================================================================|
 |    [SetMenuStrip()](autodocs-2.0/intuition-library-setmenustrip.md)  Set a menu for an open window.                     |
 |  [ClearMenuStrip()](autodocs-2.0/intuition-library-clearmenustrip.md)  Clear the menu of an open window.                  |
 |  [ResetMenuStrip()](autodocs-2.0/intuition-library-resetmenustrip.md)  Set a pre-calculated menu for an open window.      |
 |-----------------------------------------------------------------------|
 |     [ItemAddress()](autodocs-2.0/intuition-library-itemaddress.md)  Find the address of a menu item from its position. |
 |-----------------------------------------------------------------------|
 |         [OffMenu()](autodocs-2.0/intuition-library-offmenu.md)  Disable a menu in a menu strip.                    |
 |          [OnMenu()](autodocs-2.0/intuition-library-onmenu.md)  Enable a menu in a menu strip.                     |
 |_______________________________________________________________________|
