> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The memory information routines [AvailMem()](libraries/20-memory-info-functions-calling-memory-information.md) and [TypeOfMem()](libraries/20-memory-info-functions-calling-memory-information.md) can provide the
amount of memory available in the system, and the attributes of a
particular block of memory.
