# 4 Intuition Windows / Window Attributes

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This section discusses all window attributes.  As mentioned earlier, a
window's attributes may be specified with either [TagItems](libraries/4-intuition-windows-basic-window-structures-and-functions.md), [NewWindow](libraries/4-intuition-windows-basic-window-structures-and-functions.md) or
[ExtNewWindow](libraries/4-intuition-windows-basic-window-structures-and-functions.md) depending on how the window is opened.

Attributes are listed here by their [TagItem](libraries/4-intuition-windows-basic-window-structures-and-functions.md) ID name (TagItem.ti_Tag).  For
each tag item, the equivalent field setting in the [NewWindow](libraries/4-intuition-windows-basic-window-structures-and-functions.md) structure is
also listed if it exists.  Some window attributes specified with tags are
available only in Release 2 and have no NewWindow equivalent.
