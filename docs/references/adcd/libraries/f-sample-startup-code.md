# F Sample Startup Code

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This appendix contains the source code to build Startup code modules.
The files include:

 [lmkfile](libraries/lib-startup-lmkfile.md)             A makefile to build the startup code modules.
 [startup.asm](libraries/lib-startup-startup-asm.md)         Assembler source for startup modules
 [allstartup.i](libraries/lib-startup-allstartup-i.md)        Main include file
               __
 [Astartup.i](libraries/lib-startup-astartup-i.md)      |
 [AWstartup.i](libraries/lib-startup-awstartup-i.md)     |
 [Qstartup.i](libraries/lib-startup-qstartup-i.md)      |-- Include files for specific startup modules
 [Rstartup.i](libraries/lib-startup-rstartup-i.md)      |
 [RWstartup.i](libraries/lib-startup-rwstartup-i.md)     |
 [RXstartup.i](libraries/lib-startup-rxstartup-i.md)   __|

The object code for the above include files can be found in the
Reference_Library/Lib_Startup drawer on this disc.
