# B Boopsi Class Reference / itexticlass

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Class:                  itexticlass
Superclass:             [imageclass](libraries/b-boopsi-class-reference-imageclass.md)
Include File:           <intuition/[imageclass.h](autodocs-2.0/includes-intuition-imageclass-h.md)>

This is a class of image objects that render an [IntuiText](libraries/8-creating-text-intuitext-structure.md) structure.
Using some of the [imageclass](libraries/b-boopsi-class-reference-imageclass.md) attributes, the object can override some of
the parameters in the IntuiText structure.  This class makes it easy to
share an IntuiText structure between objects.
