# B / sysiclass / Attributes: SYSIA_Which (I)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute identifies which of the system image glyphs the [sysiclass](libraries/b-boopsi-class-reference-sysiclass.md)
object uses.  It can be one of the 11 glyphs [described above](libraries/b-boopsi-class-reference-sysiclass.md).
