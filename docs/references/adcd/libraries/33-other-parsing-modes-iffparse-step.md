# 33 / / Other Parsing Modes / IFFPARSE_STEP

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This mode is identical to [IFFPARSE_RAWSTEP](libraries/33-other-parsing-modes-iffparse-rawstep.md), except that, before control
returns to your code, the chunk handler (if any) for the chunk is invoked.
