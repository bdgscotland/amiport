# 27 / / The RastPort Structure / Text Attributes

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Text attributes and font information are stored in the [RastPort](libraries/27-drawing-routines-the-rastport-structure.md) fields
[Font](libraries/27-drawing-routines-the-rastport-structure.md), [AlgoStyle](libraries/27-drawing-routines-the-rastport-structure.md), [TxFlags](libraries/27-drawing-routines-the-rastport-structure.md), [TxHeight](libraries/27-drawing-routines-the-rastport-structure.md), [TxWidth](libraries/27-drawing-routines-the-rastport-structure.md), [TxBaseline](libraries/27-drawing-routines-the-rastport-structure.md) and [TxSpacing](libraries/27-drawing-routines-the-rastport-structure.md).
These are normally set by calls to the graphics font routines which are
covered separately in the chapter on "[Graphics Library and Text](libraries/29-graphics-library-and-text.md)".
