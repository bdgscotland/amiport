# B / gadgetclass / Attributes: GA_SysGType (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute corresponds to the system gadget type portion of the
gadget's [Gadget.GadgetType](libraries/5-intuition-gadgets-gadget-structure.md) fields.  This attribute is any one of the
following flags (from <intuition/[intuition.h](autodocs-2.0/includes-intuition-intuition-h.md)>):


    GTYP_SIZING, GTYP_WDRAGGING, GTYP_SDRAGGING, GTYP_WUPFRONT,
    GTYP_SUPFRONT, GTYP_WDOWNBACK, GTYP_SDOWNBACK, GTYP_CLOSE
See the "[Intuition Gadgets](libraries/5-intuition-gadgets-gadget-structure.md)" chapter of the Amiga ROM Kernel Reference
Manual: Libraries for more information.
