# B / groupgclass / Changed Methods: OM_REMMEMBER

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This method removes a gadget added to the group object's list with
OM_ADDMEMBER.  Note that all members of the [groupgclass](libraries/b-boopsi-class-reference-groupgclass.md) object will be
deleted by [OM_DISPOSE](libraries/b-groupgclass-changed-methods-om-dispose.md).
