# B / propgclass / Attributes: PGA_NewLook (I)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This is a boolean attribute which corresponds to the [PROPNEWLOOK](libraries/5-initialization-of-proportional-gadget-the-propinfo.md) flag
[PropInfo](libraries/5-initialization-of-proportional-gadget-the-propinfo.md) structure's [Flags](libraries/5-initialization-of-proportional-gadget-the-propinfo.md) field (defined in <intuition/[intuition.h](autodocs-2.0/includes-intuition-intuition-h.md)>).  If
this attribute is TRUE, the new [propgclass](libraries/b-boopsi-class-reference-propgclass.md) object will use Release 2
imagery rather than the Release 1.3 imagery.
