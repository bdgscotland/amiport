# B / propgclass / Attributes: PGA_Freedom (IG)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute tells a [propgclass](libraries/b-boopsi-class-reference-propgclass.md) object on which axis the gadget's knob
is free to move, the horizontal or the vertical.  It is either [FREEHORIZ](libraries/5-initialization-of-proportional-gadget-the-propinfo.md)
or [FREEVERT](libraries/5-initialization-of-proportional-gadget-the-propinfo.md).  The default is FREEVERT.
