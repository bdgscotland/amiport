# B / imageclass / Attributes: IA_Data (ISG)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

A pointer to general image data.  This value is stored in the [ImageData](libraries/8-creating-images-image-structure.md)
field of the [Image](libraries/8-creating-images-image-structure.md) structure.
