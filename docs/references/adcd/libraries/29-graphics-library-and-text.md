# 29 Graphics Library and Text

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

On the Amiga, rendering text is similar to rendering lines and shapes.
The Amiga graphics library provides text functions based around the
[RastPort](libraries/27-drawing-routines-the-rastport-structure.md) structure, which makes it easy to intermix graphics and text.
