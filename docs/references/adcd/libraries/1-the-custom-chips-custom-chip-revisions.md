# 1 / / The Custom Chips / Custom Chip Revisions

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The [custom chips](libraries/1-programming-in-the-amiga-environment-the-custom-chips.md) have been revised as the Amiga platform has evolved and
newer models of the Amiga developed.  The latest revision of the Amiga
custom chips is known as the Enhanced Chip Set, or [ECS](libraries/27-introduction-about-ecs.md).  Certain features
of the Amiga operating system, such as higher resolution screens and
[special genlock modes](libraries/27-graphics-primitives-ecs-and-genlocking-features.md), require the ECS version of the custom chips.  In
this book, features that require ECS are noted in the accompanying text.
For more details about the special features of ECS, see [Appendix C](hardware/amiga-hardware-reference-manual-c-enhanced-chip-set.md) of
the Amiga Hardware Reference Manual.
