# B / gadgetclass / Attributes: GA_GZZGadget, GA_SysGadget (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These are boolean attributes that correspond to the flags in the object's
[Gadget.GadgetType](libraries/5-intuition-gadgets-gadget-structure.md) field.  If the value passed with the attribute is TRUE,
the corresponding flag in Gadget.GadgetType is set.  If the value passed
with the attribute is FALSE, the corresponding flag in Gadget.GadgetType
is cleared.  See the <intuition/[intuition.h](autodocs-2.0/includes-intuition-intuition-h.md)> include file or the
"[Intuition Gadgets](libraries/5-intuition-gadgets-gadget-structure.md)" chapter of the Amiga ROM Kernel Reference Manual:
Libraries for more information.
