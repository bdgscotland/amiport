# B / frbuttonclass / Changed Methods: GM_RENDER

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

For this method, the [frbuttonclass](libraries/b-boopsi-class-reference-frbuttonclass.md) object first draws the framing image by
sending the image an [IM_DRAWFRAME](libraries/b-imageclass-new-methods-im-drawframe.md) message.  The object then draws its
display element.
