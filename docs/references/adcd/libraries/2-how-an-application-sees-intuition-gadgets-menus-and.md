# 2 / How an Application Sees Intuition / Gadgets, Menus And Requesters

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Intuition [screens and windows](libraries/2-how-an-application-sees-intuition-screens-and-windows.md) provide an orderly way for multiple programs
to share the display and input devices.  Each application also needs a
method for the user to send commands to it and select its options.
Intuition supplies gadgets, menus and requesters for this purpose.
