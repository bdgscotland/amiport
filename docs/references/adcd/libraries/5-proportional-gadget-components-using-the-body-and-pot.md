# 5 / / Proportional Gadget Components / Using the Body and Pot Values

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The body and pot values of a [proportional](libraries/5-intuition-gadgets-proportional-gadget-type.md) gadget are "Intuition friendly"
numbers, in that they represent concepts convenient to Intuition, and not
to the application.  The application must translate these numbers to
internal values before acting on them.
