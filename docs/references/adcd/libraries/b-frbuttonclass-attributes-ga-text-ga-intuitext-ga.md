# B / frbuttonclass / Attributes:GA_Text,GA_IntuiText,GA_LabelImage (IS)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

These attributes tell the [frbuttonclass](libraries/b-boopsi-class-reference-frbuttonclass.md) object what kind of imagery to use
as its display element.  See their description in the [gadgetclass](libraries/b-boopsi-class-reference-gadgetclass.md) entry
for more information.
