# 12 / OOPOverview / The Public Classes

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Intuition contains 14 public classes, all of which are descendants of the
[rootclass](libraries/b-boopsi-class-reference-rootclass.md).  There are three primary classes that descend directly from
rootclass: imageclass, gadgetclass, and [icclass](libraries/12-boopsi-object-oriented-intuition-oop-overview.md).
