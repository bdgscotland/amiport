# B / imageclass / Attributes: IA_Pens ()

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute points to an alternative pen array for the image.
[Imageclass](libraries/b-boopsi-class-reference-imageclass.md) does not support this attribute, it is described here for
subclasses to use.  See the "[Intuition Screens](libraries/3-intuition-screens-drawinfo-and-the-3d-look.md)" chapter of the Amiga ROM
Kernel Reference Manual: Libraries for more information on the pen array.
