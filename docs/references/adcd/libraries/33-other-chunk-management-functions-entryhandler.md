# 33 / / Other Chunk Management Functions / EntryHandler()

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This is used to install your own custom chunk entry handler.  See the
"[Custom Chunk Handlers](libraries/33-advanced-topics-custom-chunk-handlers.md)" section below for more information.  [StopChunk()](libraries/33-controlling-parsing-stopchunk.md),
[PropChunk()](libraries/33-controlling-parsing-propchunk-findprop.md), and [CollectionChunk()](libraries/33-management-functions-collectionchunk-and-findcollection.md) are internally built on top of this.
