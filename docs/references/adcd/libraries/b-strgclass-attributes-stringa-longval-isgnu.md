# B / strgclass / Attributes: STRINGA_LongVal (ISGNU)

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This attribute tells [strgclass](libraries/b-boopsi-class-reference-strgclass.md) that this gadget is an integer string
gadget and the new value of the integer is this attribute's value.
