# B / groupgclass / Changed Methods: OM_SET

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This method passes most attributes to the superclass, but propagates
changes in position to its members appropriately. Also, [GA_Width](libraries/b-gadgetclass-attributes-ga-left-ga-top-ga-width-ga-height.md) and
[GA_Height](libraries/b-gadgetclass-attributes-ga-left-ga-top-ga-width-ga-height.md) are calculated from the position and dimension of the membership.
