# B / frameiclass / Changed Methods: IM_FRAMEBOX

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This method asks a [frameiclass](libraries/b-boopsi-class-reference-frameiclass.md) image what its dimensions would be if it
has to frame a specific rectangular area.  See the [imageclass](libraries/b-boopsi-class-reference-imageclass.md) description
for [IM_FRAMEBOX](libraries/b-imageclass-new-methods-im-framebox.md) for more information.
