# 8 Interface Hardware / Serial I/O Interface

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

A 25-pin connector on the back panel of the computer serves as the general
purpose  [serial interface](hardware/e-explanation-of-i-o-connectors-serial-interface.md) . This connector can drive a wide range of
different peripherals, including an external modem or a serial printer.

For pin connections, see  [Appendix E](hardware/amiga-hardware-reference-manual-e-i-o-connectors-and.md) .
