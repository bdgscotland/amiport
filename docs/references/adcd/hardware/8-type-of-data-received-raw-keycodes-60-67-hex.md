# 8 / / Type Of Data Received / RAW Keycodes -> 60-67 hex

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Key codes for qualifier keys

   60      Left Shift
   61      Right Shift
   62      Caps Lock
   63      Control
   64      Left Alt
   65      Right Alt
   66      Left Amiga (or Commodore key)
   67      Right Amiga
