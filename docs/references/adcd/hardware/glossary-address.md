# Glossary / address

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

A byte-numbered memory location.  The Zorro II bus is based on a 24-bit
address, the Zorro III bus on a 32-bit address.
