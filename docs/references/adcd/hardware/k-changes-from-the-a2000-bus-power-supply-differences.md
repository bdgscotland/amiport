# K / / Changes from the A2000 Bus / Power Supply Differences

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The Zorro II bus is defined as supplying  [+5VDC](hardware/k-signal-description-power-connections.md)  @ 2 Amps to each slot,
with one slot per backplane supplying 5.0VDC @ 4.0 Amps.  The Zorro III bus
only provides the 5.0VDC @ 2.0 Amps for each slot.
