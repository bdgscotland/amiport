# Amiga® Hardware Reference Manual: 4 Sprite Hardware

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

This chapter discusses sprites which are special graphic objects that are
easy to define and easy to animate.  The following sprite topics are
covered:

   *  Defining the size, shape, color, and screen position of sprites.

   *  Displaying and moving sprites.

   *  Combining sprites for more complex images, additional width, or

      additional colors.
   *  Reusing a sprite DMA channel multiple times within a display field

      to create more than eight sprites on the screen at one time.
