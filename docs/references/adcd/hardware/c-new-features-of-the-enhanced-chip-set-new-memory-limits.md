# C / New Features of the Enhanced Chip Set / New Memory Limits

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The A3000 has 1 MB of Chip memory, and with proper jumpering of the
motherboard, an additional 1 MB can be added.  On the A2000, the enhanced
Agnus can access up to 1 megabyte of Chip memory with proper jumpering of
the motherboard.  This provides programs with more blitter-accessible
memory for animation and graphics applications.
