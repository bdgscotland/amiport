# F 8520 Complex Interface Adapters / Control Registers

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

There are two control registers in the 8520, CRA and CRB.  CRA is
associated with  [Timer A](hardware/f-register-functional-description-interval-timers-timer-a-b.md)  and CRB is associated with  [Timer B](hardware/f-register-functional-description-interval-timers-timer-a-b.md) .  The
format of the registers is as follows:
