# K / Electrical Specifications / Temperature Range

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The Zorro III bus is specified for operation over a temperature range
of 0 degrees Celsius to 70 degrees Celsius.
