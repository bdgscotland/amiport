# 7 / Interrupts / Maskable Interrupts

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

[Interrupt levels 1 through 6](hardware/7-setting-and-clearing-bits-figure-7-4-interrupt-priorities.md)  are generated. Control registers within the
peripherals chip allow you to mask certain of these sources and prevent
them from generating a 680x0 interrupt.
