# K Zorro Expansion Bus / Signal Description

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The signals detailed here are the Zorro III mode signals.  While some of
this information is the same as in the Zorro II  [signal description](hardware/k-zorro-ii-compatibility-signal-description.md)  in the
Zorro II Compatibility section, many bus signals that seem alike behave
differently in Zorro III mode than Zorro II mode.  These can be a very
important differences; thus the complete set of signals is detailed here.
