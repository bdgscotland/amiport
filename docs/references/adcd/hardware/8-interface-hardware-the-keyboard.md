# 8 Interface Hardware / The Keyboard

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The keyboard is interfaced to the system via the serial  [shift register](hardware/f-8520-complex-interface-adapters-serial-shift-register-sdr.md) 
on one of the 8520 CIA chips.  The keyboard data line (KDAT) is connected
to the  [SP pin](hardware/e-connectors-and-interfaces-port-signal-assignments-for.md) , the keyboard clock (KCLK) is connected to the  [CNT pin](hardware/e-connectors-and-interfaces-port-signal-assignments-for.md) .
 [Appendix G](hardware/amiga-hardware-reference-manual-g-keyboard-interface.md)  contains a full description of the interface.
