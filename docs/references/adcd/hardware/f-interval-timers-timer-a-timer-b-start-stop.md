# F / / Interval Timers (Timer A, Timer B) / Start/Stop

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

A control bit (CRx0) allows the timer to be started or stopped by the
microprocessor at any time.
