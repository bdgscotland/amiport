# C / New Features of the Enhanced Chip Set / New Mode Resolutions

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The enhanced Denise chip provides the new  [SuperHires mode](hardware/c-ecs-hardware-and-the-graphics-library-superhires-mode.md)  with up to 1280
horizontal pixels per scanline on a standard NTSC or PAL display.

All of the standard display resolutions and depths of the original chip
set are supported with the ECS.
