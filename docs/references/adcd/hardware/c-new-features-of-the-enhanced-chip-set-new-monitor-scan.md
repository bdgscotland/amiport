# C / New Features of the Enhanced Chip Set / New Monitor Scan Rates

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

The V2.0 Kickstart and ECS chips support a new high resolution
Productivity mode.  With the addition of a  [multi-sync monitor](hardware/c-ecs-hardware-and-graphics-library-multi-sync-and-bi-sync.md) , this mode
allows 640 x 480, non-interlaced screens in up to four colors. All
programs which open and operate in the Workbench screen will automatically
benefit from Productivity text and graphics. In addition, new programs can
open their own Productivity screens in a system standard fashion.
