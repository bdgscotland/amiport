# Glossary / ROM

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Short for read-only memory.  ROM is the part of the Amiga's memory which
is permanent, or non-volatile.  The Amiga's operating system is stored in
ROM.
