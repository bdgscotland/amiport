# A Register Summary / AUDxLEN

                          Agnus/
                  Read/   Denise/
Register Address  Write   Paula         Function
-------- -------  -----   -------       --------
AUDxLEN    0A4      W       P      Audio channel x length


```c
                 This register contains the length (number of words) of
                 audio channel x DMA data.
```
