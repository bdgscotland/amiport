# 3 / Advanced Topics / Interactions Among Playfields and Other Objects

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

Playfields share the display with sprites. Chapter 7, System Control
Hardware, shows how playfields can be given different
 [video display priorities](hardware/7-system-control-hardware-video-priorities.md)  relative to the sprites and how playfields can
 [collide](hardware/7-system-control-hardware-collision-detection.md)  with (overlap) the sprites or each other.
