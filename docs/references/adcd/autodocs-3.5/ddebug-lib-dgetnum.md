# ddebug.lib/DGetNum

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

   NAME
	DGetNum - get a number from the parallel port

   SYNOPSIS
	number = DGetNum()
	D0

   FUNCTION
	get a signed decimal integer from the parallel port.
