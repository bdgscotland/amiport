# ddebug.lib/KCmpStr

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

   NAME
	KCmpStr - compare two null terminated strings

   SYNOPSIS
	mismatch = KCmpStr(string1, string2)
	D0		  A0	   A1

   FUNCTION
	string1 is compared to string2 using the ASCII collating
	sequence.
