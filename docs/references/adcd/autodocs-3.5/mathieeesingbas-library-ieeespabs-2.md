   NAME
	IEEESPAbs -- compute absolute value of IEEE single precision argument

   SYNOPSIS
	  x   = IEEESPAbs(  y  );
	 d0		    d0

	float	x,y;

   FUNCTION
	Take the absolute value of argument y and return it to caller.

   INPUTS
	y -- IEEE single precision floating point value
