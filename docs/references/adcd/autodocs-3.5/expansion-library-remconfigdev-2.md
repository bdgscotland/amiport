# expansion.library/RemConfigDev

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

   NAME
	RemConfigDev - remove a [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure from the system

   SYNOPSIS
	RemConfigDev( configDev )
	              A0

   FUNCTION
	(Not typically called by user code)

	This routine removes the specified [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure from the
	list of Configuration Devices in the system.

   INPUTS
	configDev - a valid [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure.
