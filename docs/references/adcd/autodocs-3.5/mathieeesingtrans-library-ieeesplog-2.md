   NAME
	IEEESPLog -- compute the natural logarithm of a floating point number

   SYNOPSIS
	  x   = IEEESPLog(  y  );
	d0	          d0

	float	x,y;

   FUNCTION
	Compute ln(y) in IEEE single precision

   INPUTS
	y - IEEE single precision floating point value
