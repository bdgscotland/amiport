# expansion.library/AddConfigDev

> *Source: Amiga Developer CD v2.1. (C) Commodore-Amiga / Amiga Inc.*

   NAME
	AddConfigDev - add a new [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure to the system

   SYNOPSIS
	AddConfigDev( configDev )
	              A0

   FUNCTION
	(Not typically called by user code)

	This routine adds the specified [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure to the
	list of Configuration Devices in the system.

   INPUTS
	configDev - a valid [ConfigDev](autodocs-3.5/include-libraries-configvars-h.md) structure.
