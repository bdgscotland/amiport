What is this?

Unix/POSIX tools we're evaluating for porting to AmigaOS 3.x. Each candidate is scored on how ready it is to port based on its source complexity and which POSIX functions it needs. Ready means the tool can be ported now with existing shim functions. Feasible means it needs minor work. Blocked means it depends on POSIX features we haven't implemented yet (like fork or popen). Click any row to see the full analysis.

Hardware column shows compatibility: • fits, • tight, • won't fit. Three dots = stock A1200 / A1200 + accelerator / A4000.
