Getting Started

Download and install tools on your Amiga.

  1. Browse packages Find what you need on the Packages page.
  2. Download Grab the .lha archive. Transfer to your Amiga via network, CF card, or serial.
  3. Install Extract the archive to your system: lha x grep-1.68.lha SYS:
Coming soon: amiget — a CLI package manager for AmigaOS. amiget install grep will handle downloads and installation automatically. Requires a TCP/IP stack (Miami, AmiTCP, or Roadshow).
About amiport

amiport is an AI-powered pipeline that ports POSIX/Unix tools to the Commodore Amiga. It analyzes C source code, transforms POSIX system calls to AmigaOS equivalents, cross-compiles for 68020+, and tests on a virtual Amiga.

  1. Analyze Scan source for portability issues
  2. Transform Replace POSIX calls with Amiga equivalents
  3. Build Cross-compile with m68k-amigaos-gcc
  4. Test Verify on virtual AmigaOS (vamos + FS-UAE)
  5. Package .lha archive, Aminet readme, amiport index

The pipeline uses specialized AI agents for each stage. Source analysis, code transformation, build management, and testing are all automated. Each port produces a working Amiga binary from unmodified upstream source.
